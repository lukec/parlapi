ParlAPI.ca

What is this?

This site mirrors the content available on parl.gc.ca and re-represents the data using modern web techniques to provide a simple interface for subsequent re-use. Ideally this interface would be provided by the House of Commons directly and this site could go away. This site is considered as an experiment by it's author.

All of the Parliamentary content on this site should not be considered official, and used with respect according to the Speakers Permission and in accordance with Parliamentary Copyright and Privilege. The software that runs this site is released as Open Source Software.

Many Canadian citizens have wasted time scraping information from Parl.gc.ca because the data they need is not readily available through the existing APIs. This site aims to build a simple API for these needs so that citizens can spend that time on building innovative applications to further our democracy.
